# Administrador - Gestionar Gabinete

[Volver a Administrador](./index.md) | [Volver al indice](../index.md)

## Listar integrantes de gabinete

![Listado gabinete](../assets/admin-gabinete-listado.png)

1. Ingresar a **Gestionar Gabinete**.
2. Revisar la tabla de integrantes.
3. Usar el buscador para filtrar por nombre, usuario o email.

## Crear integrante de gabinete

![Nuevo integrante de gabinete](../assets/admin-gabinete-nuevo.png)

1. Presionar **Nuevo integrante**.
2. Completar datos del usuario.
3. Presionar **Guardar**.

## Ver detalle

![Detalle gabinete](../assets/admin-gabinete-detalle.png)

1. Seleccionar un integrante.
2. Presionar **Ver detalle**.
3. Revisar la informacion.
4. Presionar **Editar datos** si corresponde.

## Editar integrante

1. Seleccionar el integrante.
2. Presionar **Editar**.
3. Modificar los campos necesarios.
4. Dejar contrasena vacia si no se quiere cambiar.
5. Presionar **Guardar**.
