# Administrador - Configurar Materias

[Volver a Administrador](./index.md) | [Volver al indice](../index.md)

## Listar materias

![Materias](../assets/admin-materias.png)

1. Ingresar a **Configuracion**.
2. Seleccionar **Materias**.
3. Revisar la tabla de materias.

## Crear materia

1. Completar el nombre de la materia.
2. Presionar **Guardar**.

## Editar materia

1. Seleccionar una materia.
2. Modificar el nombre.
3. Presionar **Guardar**.

## Eliminar materia

1. Seleccionar una materia.
2. Presionar la accion de eliminar.
3. Confirmar la eliminacion si el sistema lo solicita.

## Validaciones esperadas

- No permite guardar una materia sin nombre.
- Muestra error si la materia no puede eliminarse por estar asociada a otros datos.
