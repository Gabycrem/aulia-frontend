## Ver roles del sistema

![Roles](../assets/admin-roles.png)

1. Ingresar a **Configuracion**.
2. Seleccionar **Roles**.
3. Revisar los roles disponibles.

## Uso de roles

Los roles definen a que panel puede acceder cada usuario:

- Administrador.
- Docente.
- Gabinete.
- Alumno.
- Directivo.
