# Assets del Manual de Usuario

[Volver al manual](../index.md)

Guardar aca las capturas usadas por el manual.

## Formato recomendado

- PNG.
- Nombres en minuscula.
- Sin espacios.
- Sin acentos.
- Separar palabras con guion medio.

Ejemplo:

```txt
admin-alumno-editar.png
```

La lista completa de capturas sugeridas esta en [Guia de capturas](../capturas.md).
