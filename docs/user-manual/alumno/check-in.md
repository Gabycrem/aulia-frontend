# Alumno - Registrar Check-in

[Volver a Alumno](./index.md) | [Volver al indice](../index.md)

## Registrar estado emocional

![Check-in alumno](../assets/alumno-dashboard.png)

1. Seleccionar como se siente hoy.
2. Seleccionar el contexto.
3. Escribir un comentario.
4. Marcar **Quiero que alguien del gabinete me contacte** si corresponde.
5. Presionar **Guardar**.

## Cancelar

1. Presionar **Cancelar**.
2. El formulario vuelve a su estado inicial.

## Validaciones esperadas

- Debe seleccionar un estado emocional.
- Debe seleccionar un contexto.
- Debe escribir un comentario.
- Si el sistema no identifica alumno o curso, muestra un mensaje y no guarda.
