# Gabinete - Agenda y Alertas fuera de alcance

[Volver a Gabinete](./index.md) | [Volver al indice](../index.md)

## Estado actual

Agenda y alertas no forman parte del alcance funcional del MVP actual.

Las opciones pueden aparecer visibles pero deshabilitadas en el menu. Si se accede por URL directa, el sistema debe mostrar una pantalla informativa y no permitir operar el modulo.

## Agenda de gabinete

La agenda de gabinete no tiene un flujo operativo completo en esta etapa.

No se deben registrar, editar ni consultar eventos desde este modulo durante el MVP.

## Alertas

El modulo de alertas queda pendiente para una etapa posterior.

El dashboard puede mostrar informacion relacionada con check-ins o pedidos de contacto, pero eso no implica que exista una gestion completa de alertas.
