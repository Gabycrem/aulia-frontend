# Errores Frecuentes

[Volver al indice](./index.md)

## Usuario o contrasena incorrectos

El sistema muestra un error y permanece en login.

Que hacer:

1. Revisar usuario.
2. Revisar contrasena.
3. Intentar nuevamente.

## Sesion vencida

El sistema puede redirigir al login cuando la sesion deja de ser valida.

Que hacer:

1. Volver a iniciar sesion.
2. Repetir la accion que se estaba realizando.

## Permiso insuficiente

Ocurre cuando un usuario intenta ingresar a una pantalla que no corresponde a su rol.

Que hacer:

1. Volver al panel propio del rol.
2. Si el acceso deberia estar permitido, informar al administrador.

## Backend no disponible

Puede aparecer un mensaje de conexion cuando el servidor no responde.

Que hacer:

1. Esperar unos segundos.
2. Reintentar la accion.
3. Si persiste, informar al equipo tecnico.

## Registro inexistente

Puede ocurrir si se intenta abrir un alumno, docente, caso o intervencion que ya no existe.

Que hacer:

1. Volver al listado.
2. Actualizar la pantalla.
3. Buscar nuevamente el registro.

## Formulario incompleto

El sistema muestra un mensaje indicando que faltan datos.

Que hacer:

1. Revisar los campos obligatorios.
2. Completar la informacion faltante.
3. Guardar nuevamente.
