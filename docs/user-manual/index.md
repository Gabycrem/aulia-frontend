## Roles cubiertos

| Rol | Que puede hacer |
| --- | --- |
| Administrador | Gestiona alumnos, docentes, gabinete, cursos, materias y roles. |
| Docente | Consulta alumnos asignados, solicita intervenciones y revisa el estado de sus solicitudes. |
| Gabinete | Gestiona derivaciones, casos e intervenciones. Agenda y alertas quedan fuera del alcance del MVP. |
| Alumno | Registra su check-in emocional. |

## Recomendacion de lectura

1. Leer primero [Acceso y cierre de sesion](./01-acceso-sesion.md).
2. Revisar [Navegacion general](./02-navegacion.md).
3. Ir al modulo correspondiente segun el rol.
4. Consultar [Errores frecuentes](./errores-frecuentes.md) si aparece un mensaje inesperado.

## Capturas

Las capturas deben guardarse en [`./assets`](./assets/README.md).

Cada archivo del manual incluye espacios sugeridos para imagenes. Si todavia no estan cargadas, el texto igual queda completo y navegable.

## Funcionalidades fuera de alcance del MVP

Las siguientes opciones pueden aparecer visibles pero deshabilitadas, o mostrar una pantalla informativa si se accede por URL directa:

- Agenda docente.
- Agenda de gabinete.
- Alertas de gabinete.
